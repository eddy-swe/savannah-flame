# Savannah Flame

A vibrant and engaging web project built with HTML and CSS, showcasing modern web design principles and responsive styling.

## 📋 Overview

Savannah Flame is a frontend project composed primarily of HTML (61.2%) and CSS (38.8%) that creates an interactive and visually appealing web experience. This repository demonstrates clean markup and sophisticated styling techniques.

## 🎨 Project Composition

- **HTML**: 61.2% - Semantic structure and content
- **CSS**: 38.8% - Styling, layout, and responsive design

## ✨ Features

- Clean and semantic HTML structure
- Modern CSS styling and animations
- Responsive design principles
- Optimized for various screen sizes

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/eddy-swe/savannah-flame.git
cd savannah-flame
```

2. Open the project in your browser:
```bash
# Simply open the index.html file in your preferred browser
open index.html
```

Or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```
